Web Development Report

Introduction

The essential story told by my portfolio site is that of a an experienced game designer in 2034 who has created impressive interactive experiences and games across multiple platforms. The portfolio showcases not only the projects I wish I would have completed by then, but also my journey through emerging technologies and innovative design processes. The audience I am addressing includes potential employers, collaborators, and perhaps most importantly, aspiring game developers looking for inspiration. The website uses a simple, minimalistic but modern structure, focusing on accessibility and ease of navigation.

Inspiration

  1. Joan – Interactive Artist: What I liked and inspired me the most is the responsive icons on her page and her simple navigation bar.
  2. Victoire Douy – Illustrator: What I liked and inspired me the most is the minimalistic, responsive and nice UI; we see just the minimal that we need to see and nothing more, unless we want to see more.
  3. Adam Sandoval – Web Designer: This one also inspired me greatly. I liked the way he uses the mouse over and cursor appearance. He also presents himself so well on his about page.

Accessibility

  • Alt Text for Images
  • Text Resize or Font Resizing:This feature helps users with visual impairments by allowing them to adjust the font size for better readability. It is accessed when the user clicks on the accessibility icon on the navigation bar.
  • High Contrast: As per the Web Content Accessibility Guidelines (WCAG) for the contrast ratios, the contrast ratio between my background color #EBBDB5 and #000000 font is 7.19:1, which meets WCAG AA standards for both normal and large text.

Learning

  • CSS Grid:I was adamant to learn this layout style because initially it was confusing especially when compared to the simple flex method, but with the use of a good book, I have mastered this layout style.
  • JavaScript Event Handling:I learned JavaScript for the first time in this course. Event handling is one of the aspects of JS that I am the most proud of.
  • Accessibility Guidelines:I spend a long time reading about the Web Content Accessibility Guidelines (WCAG), I became fascinated by how much web developers can actually help users with disabilities take the most of the web, but sadly this area is usually ignored by most web developers.

Evaluation I

Aspects that were particularly successful:

  • Design Consistency:I took time to ensure that the website design is consistent which was difficult because I had to apply same code to every page, but it turned out to be one success to be highlighted on this learning journey.
  • Responsive Behavior:Even though, I chose to go with a minimalistic style, I also wanted to be as responsive as the inspiring websites I used were. It was trial and error, but since I had a good knowledge of JS, I figured a way to do anything I had in mind.
  • Interactive Features:I wanted the works page to look appealing to a fellow game developer or enthousiast, so I gave it a game-like theme where the user has to interact with a picture to find out about the work. It was a great success.
